## Component Details

### Message Handler
**Responsibilities:**
- Receive WhatsApp webhook events
- Parse message content
- Detect commands vs. natural language
- Route to appropriate handler
- Send responses back to user

**Key Functions:**
- `handleIncomingMessage()` - Main entry point
- `processCommand()` - Command router
- `handleSetBudget()` - Budget creation
- `handleSummary()` - Generate reports
- `checkBudgetAlert()` - Alert logic

### Finance Agent
**Responsibilities:**
- Interface with Claude AI
- Extract expense information
- Provide financial insights
- Generate contextual responses

**Key Functions:**
- `processMessage()` - AI conversation
- `detectExpense()` - NLP extraction
- `getFinancialContext()` - Gather user data

### Database Layer
**Responsibilities:**
- Store user data
- Track expenses
- Manage budgets
- Provide query methods

**Key Operations:**
- CRUD for users
- CRUD for expenses
- CRUD for budgets
- Aggregation queries
